Citing GPkit
************

If you use GPkit, please cite it with the following bibtex::

    @Misc{gpkit,
          author={Edward Burnell and Warren Hoburg},
          title={GPkit software for geometric programming},
          howpublished={\url{https://github.com/hoburg/gpkit}},
          year={2016},
          note={Version 0.5.1}
         }
