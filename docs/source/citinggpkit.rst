Citing GPkit
************

If you use GPkit, please cite it with the following bibtex::

    @Misc{gpkit,
          author={MIT Department of Aeronautics and Astronautics},
          title={GPkit},
          howpublished={\url{https://github.com/convexopt/gpkit}},
          year={2015},
          note={Version 0.3.2}
         }
